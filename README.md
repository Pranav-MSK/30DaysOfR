# 📘 30 Days of R — Learning Journey
Welcome to my 30 Days of R challenge!
This repository documents my learning journey as I explore R programming, practice real-world examples, and build a habit of coding regularly.

This challenge is progress-based, not calendar-based.
I may not code every single day, but I will complete 30 meaningful learning sessions and track them here.

---

## 🎯 Goal
Learn the fundamentals of R — including syntax, data structures, data wrangling, visualization, and basic statistical analysis — by completing 30 focused learning sessions.

---

## 🗂️ Structure
Each day has its own folder containing scripts, notes, or RMarkdown files.
```pythonrepl

...
Day30/
```

Inside each folder, you’ll find:
- ```.R``` scripts (practice code)
- ```.Rmd``` notebooks (notes, examples, analyses)
- datasets (if any)
